# 抖音无水印自动下载器 (Auto Douyin Saver)

基于 Android + Kotlin + Jetpack Compose 构建的一款极简、自动化、极其强大的本地抖音无水印视频抓取工具。

## ✨ 核心特性

- **🎮 无死角自动化**  
  内置悬浮窗与无障碍服务（Accessibility Service）。当你刷到喜欢的视频时，只需点击屏幕悬浮球，脚本即可以**微秒级速度**帮你呼出分享面板、提取源链接并关闭面板。整个过程如丝般顺滑，不打断你的沉浸式刷视频体验。

- **🚀 突破系统限制**  
  在 Android 11+（直至 Android 16）严格的剪贴板后台读取限制下，独创了一套“极速透明 Activity 代理”机制，完美实现系统级无感拦截并读取剪贴板中隐秘的分享文本。

- **🎬 Live 图与多图集视频支持**  
  内置深度优化的解析引擎，不仅支持抓取常规的高清无水印视频，更能智能解构**实况照片 (Live Photo)**，一次性将所有的动态视频切片完整且纯净地下载到本地。

- **💾 跨越卸载的物理备份**  
  除了基础的 `SharedPreferences` 存储外，当授予“所有文件访问权限”后，你的下载记录将被物理保存于原生的 `Documents/DouyinSaver` 目录中。即使你卸载重装 App，所有看过的风景与记录都会在启动的第一秒被完美找回！

- **🖼️ 现代 Android UI 体验**  
  全程由 Jetpack Compose (Material Design 3) 驱动，界面清爽现代；带有一键多选、支持视频在线调用（动态适配 MIME 类型）等优雅的重度操作能力。并且配有后台下载系统通知及 App 内实时任务进度反馈。

## 🛠 开发技术栈

- **语言**: Kotlin
- **界面**: Jetpack Compose 响应式布局
- **异步处理**: Coroutines (协程) + Flow
- **系统接口**: AccessibilityService / DownloadManager / BroadcastReceiver
- **存储**: SharedPreferences + 外部存储 (Scoped Storage)

## 📌 安装与使用指南

1. **授予权限**：
   在应用中首先授予 **“悬浮窗权限”** 和 **“无障碍服务”**，此为自动提取链接的必要权限。
2. **所有文件权限 (可选项)**：
   Android 11+ 若想体验“卸载防丢记录”功能，可在控制中心授权 `MANAGE_EXTERNAL_STORAGE`。
3. **刷且下载**：
   打开抖音，看到感兴趣的视频或 Live，点击屏幕右侧的悬浮“截取”图标。稍等片刻，底部会提示解析并随后发送“✅ XXX下载完成”的系统弹窗！
